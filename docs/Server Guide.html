Server Guide

You need to install node.js - see https://github.com/joyent/node. E.g.  clone from https://github.com/LearnBoost/socket.io.git, (Linux) 
./configure
make
make install
In the server directory (!) you need to install socket.io (https://github.com/LearnBoost/socket.io) - the following will do the download and install using node (installed above):
cd ubiserverjs/server

npm install socket.io

npm install connect

npm install cryptojs

Then you should be able to run the server (in the same directory):

node server.js

Note: depending on how node.js is built/installed the runtime may be called 'nodejs' rather than 'node'.
