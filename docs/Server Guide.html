Server Guide

You need to install node.js - see https://github.com/joyent/node. E.g.  clone from https://github.com/LearnBoost/socket.io.git, (Linux)
./configure
make
make install
Then you need to install socket.io (https://github.com/LearnBoost/socket.io) - the following will do the download and install using node (installed above):
npm install socket.io

Then you should be able to run the server, in ../server/

nodejs server.js
